1. Overview

QRVideo ("the App") provides video recording and management services for the shipping box packaging process. We prioritize user privacy protection and clearly explain our data handling practices in this policy.

✅ Core Principle: QRVideo does not transmit any personal information externally. All data exists only on the user's device.

2. Information We Collect

QRVideo collects minimal information, and all information is entirely under user control.

2.1 Collected Information

Information Type Collected Storage Location Description
Video Files ✅ Required Local Device Packaging videos recorded by user (user-controlled)
QR/Barcode Data ✅ Required Local Device Scanned tracking/tag codes
Metadata ✅ Optional Local Device Recording time, notes, status information
User Profile ✅ Optional Local Device Name, team, ID (optional input)
App Settings ✅ Required Local Device Resolution, frame rate, and other preferences
Location Information ❌ Not Collected - No GPS/location data collection
Contact Information ❌ Not Collected - No automatic phone/email collection
Device ID ❌ Not Collected - No device tracking (UDID, IMEI, etc.)
Third-party Tracking ❌ Not Collected - No ad tracking or analytics services

3. How We Collect Information

3.1 Explicit User Consent

  • Camera Permission: Access only when user enables "Settings" > "Camera"
  • File Saving: Save only when user clicks record start button
  • Metadata: Only information user directly inputs is saved

3.2 No Automatic Collection

  • QRVideo does not track user activity in the background
  • Data is not collected after app closes
  • Data is not transmitted to external servers
  • No cookies, tracking pixels, or beacons are used

4. Use of Information

4.1 Information Use Purposes

  • ✅ App Functionality: Video recording, editing, playback
  • ✅ Data Management: Video sorting, searching, filtering
  • ✅ User Experience: Save preferences (resolution, language, etc.)
  • ✅ Local Backup: Optional local storage backup

4.2 Prohibited Uses

  • ❌ Ad targeting
  • ❌ Third-party sales
  • ❌ Profiling or tracking
  • ❌ Marketing purposes
  • ❌ Machine learning/AI model training

5. Information Security

🔐 Security Level: All data stored on user device ensures maximum security.

5.1 Physical Security

  • All data stored only on user's local storage (SSD, HDD)
  • Not stored on cloud servers
  • No internet transmission occurs

5.2 Encryption (Optional)

  • Users can enable file encryption in Settings
  • When enabled, video files encrypted with AES-256
  • Encryption keys stored only on user's device, inaccessible externally

5.3 Access Control

  • Depends on user device's OS permission management
  • QRVideo operates only within granted permissions
  • Device lock/biometric security applies

6. Data Storage and Deletion

6.1 Data Retention Period

  • All data retained on user device until user deletes
  • No automatic deletion policy; user has full control
  • Optional: Set "auto-delete rules" in Settings for old files

6.2 Data Deletion Methods

  • Individual File: Click delete button for each clip in Videos tab
  • Bulk Delete: Select date range and delete all
  • All Data: Settings > "Delete All Data"
  • App Removal: Remove app to delete all related data

7. Third-party Sharing

🔒 Third-party Sharing Policy: QRVideo does not share any information with third parties without user consent.

7.1 Non-shareable Information

  • Video files
  • QR/barcode data
  • User profile information
  • Metadata
  • App usage history

7.2 User-controlled Sharing

Sharing only possible when user actively chooses:

  • Manual Export: User exports and shares files
  • Share Link: User-generated share links
  • Cloud Upload: User uploads to chosen cloud service

8. User Rights

8.1 Access Rights

Users can access all their data anytime through QRVideo's Videos tab.

8.2 Modification Rights

  • Edit metadata (notes, status, etc.)
  • Modify user profile information
  • Change app settings

8.3 Deletion Rights

  • Delete individual video files
  • Delete data from specific date range
  • Delete all data
  • Delete all related data when removing app

9. Policy Changes

9.1 Change Notification

  • This privacy policy may change anytime
  • Major changes communicated with app updates
  • This page's "Last Updated" date updates with changes
