1. 개요

QRVideo("이하 '앱'"이라 함)는 배송 박스 포장 과정의 비디오 기록 및 관리 서비스를 제공합니다. 우리는 사용자의 개인정보 보호를 최우선으로 생각하며, 본 정책에서 우리의 개인정보 처리 방침을 명확히 설명합니다.

✅ 핵심 원칙: QRVideo는 어떠한 개인정보도 외부로 전송하지 않습니다. 모든 데이터는 사용자의 디바이스에만 존재합니다.

2. 수집하는 정보

QRVideo는 최소한의 정보만 수집하며, 모든 정보는 전적으로 사용자 제어 하에 있습니다.

2.1 수집되는 정보

정보 유형 수집 여부 저장 위치 설명
비디오 파일 ✅ 필수 로컬 디바이스 사용자가 녹화한 포장 영상 (사용자 제어)
QR/바코드 데이터 ✅ 필수 로컬 디바이스 스캔한 송장번호/태그 코드
메타데이터 ✅ 선택 로컬 디바이스 녹화 시간, QR/바코드 정보
위치 정보 ❌ 수집 안 함 - GPS/위치 데이터 수집 없음
연락처 정보 ❌ 수집 안 함 - 전화번호, 이메일 자동 수집 없음
기기 고유 ID ❌ 수집 안 함 - UDID, IMEI 등 기기 추적 정보 수집 없음
서드파티 추적 ❌ 수집 안 함 - 광고 추적, 분석 서비스 이용 없음

3. 정보 수집 방식

3.1 명시적 사용자 동의

  • 카메라 권한: 사용자가 "설정" > "카메라"를 활성화할 때만 카메라 접근
  • 파일 저장: 사용자가 녹화 시작 버튼을 누를 때만 파일 저장
  • 메타데이터: 사용자가 직접 입력한 정보만 저장

3.2 자동 수집 없음

  • QRVideo는 배경에서 사용자 활동을 추적하지 않습니다
  • 앱이 종료된 후에도 정보를 수집하지 않습니다
  • 외부 서버로 데이터를 전송하지 않습니다
  • 쿠키, 트래킹 픽셀 또는 비콘을 사용하지 않습니다

4. 정보의 사용

4.1 정보 사용 목적

  • ✅ 앱 기능 제공: 비디오 녹화, 편집, 재생
  • ✅ 데이터 관리: 비디오 정렬, 검색, 필터링
  • ✅ 로컬 백업: 선택적 로컬 저장소 백업

4.2 정보 사용 금지 사항

  • ❌ 광고 타겟팅
  • ❌ 제3자 판매
  • ❌ 프로파일링 또는 개인화 추적
  • ❌ 마케팅 목적 사용
  • ❌ 기계학습/AI 모델 훈련용 데이터 사용

5. 정보 보안

🔐 보안 수준: 모든 데이터가 사용자 디바이스에만 저장됩니다.

5.1 물리적 보안

  • 모든 데이터는 사용자의 디바이스에만 보관됩니다
  • 앱자체적으로 클라우드 서버에 저장되지 않습니다
  • 앱자체적으로 인터넷 전송이 발생하지 않습니다
  • 사용자가 비디오 공유시 선택된 방법으로 공유 또는 전송 됩니다.

5.2 접근 제어

  • 사용자 기기의 운영체제 권한 관리에 의존합니다
  • QRVideo는 사용자가 부여한 권한 범위 내에서만 작동합니다
  • 기기 화면 잠금/생체 인식 등의 기본 보안 기능이 적용됩니다

5.3 보안 취약점 대응

  • QRVideo는 정기적으로 보안 업데이트를 제공합니다
  • 보안 이슈 발견 시 즉시 패치를 배포합니다
  • 사용자는 항상 최신 버전 사용을 권장합니다

6. 데이터 저장 및 삭제

6.1 데이터 저장 기간

  • 모든 데이터는 사용자가 삭제할 때까지 사용자의 디바이스에 보관됩니다
  • 자동 삭제 정책은 없으며, 사용자가 전적으로 제어합니다

6.2 데이터 삭제 방법

  • 대량 삭제: 날짜 범위 선택 후 일괄 삭제
  • 전체 데이터 삭제: Settings > "모든 데이터 삭제"
  • 앱 제거: 앱 제거 시 모든 관련 데이터 삭제 (기기 설정에서 "저장공간 정리" 옵션)

6.3 삭제된 데이터 복구 불가

삭제된 파일은 복구할 수 없습니다. 중요한 파일은 삭제 전 백업하시기 바랍니다.

7. 제3자 공유

🔒 제3자 공유 정책: QRVideo는 사용자의 동의 없이 어떠한 정보도 제3자와 공유하지 않습니다.

7.1 공유 불가 정보

  • 비디오 파일
  • QR/바코드 데이터
  • 사용자 프로필 정보
  • 메타데이터

7.2 사용자 제어 공유

사용자가 직접 선택한 경우에만 공유가 가능합니다:

  • 수동 내보내기: 사용자가 직접 파일을 내보내기하여 공유
  • 링크 생성: 사용자가 생성한 공유 링크를 통한 공유
  • 클라우드 업로드: 사용자가 선택한 클라우드 서비스(Google Drive, Dropbox 등)에 업로드

7.3 서드파티 서비스

  • QRVideo는 서드파티 분석 서비스(Google Analytics, Firebase 등)를 사용하지 않습니다
  • 광고 네트워크와 연동되어 있지 않습니다
  • 소셜 미디어 플러그인을 포함하지 않습니다

8. 사용자 권리

8.1 액세스 권리

사용자는 언제든지 자신의 모든 데이터에 접근할 수 있습니다. QRVideo의 Videos 탭에서 모든 녹화된 파일을 확인할 수 있습니다.

8.2 수정 권리

  • 메타데이터 (메모, 상태 등) 편집 가능
  • 사용자 프로필 정보 수정 가능
  • 앱 설정 변경 가능

8.3 삭제 권리

  • 개별 비디오 파일 삭제
  • 특정 기간의 데이터 일괄 삭제
  • 전체 데이터 삭제
  • 앱 제거 시 모든 관련 데이터 삭제

9. 정책 변경

9.1 변경 공지

  • 본 개인정보처리방침은 언제든지 변경될 수 있습니다
  • 중요한 변경 사항은 앱 업데이트 시 사용자에게 공지됩니다
  • 변경 사항이 있을 경우 이 페이지의 "최종 수정일"이 업데이트됩니다

9.2 사용자 동의

앱을 계속 사용함으로써 변경된 정책에 동의한 것으로 간주됩니다. 정책에 동의하지 않으면 앱 사용을 중단하십시오.
