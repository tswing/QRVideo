Smart Video Recording Solution for Shipping Package Preparation

📦 Shipping Verification 🏷️ QR/Barcode Tagging 📹 Auto Recording

What is QRVideo?

QRVideo is a frontend web app that efficiently records and manages the shipping box packaging process for online shopping malls.

🎯 Core Features

📷 Real-time Recording

Record the shipping box packaging process in real-time through your webcam.

🏷️ Tag-based Segmentation

Automatically segment video from QR code/barcode scan to the next tag.

📹 Clip Extraction

Extract and share video clips from desired segments.

💾 Secure Storage

Systematically manage all recordings and access them anytime.

❓ Why QRVideo?

🛡️ Dispute Prevention: Provides objective video evidence against claims of shipping damage from problematic customers.
⏱️ Time Saving: Videos are automatically saved and organized instead of manual record-keeping.
📊 Data Management: Systematic management is possible through metadata linked to tracking numbers.

📷 Camera Tab

Monitor and record the shipping box packaging process through real-time camera stream.

Key Features

  • Real-time Stream: Live video feed from webcam
  • Record Start/Stop: One-click recording control
  • QR/Barcode Scan: Tag recognition and automatic marking
  • Display Information: Current status, elapsed time, tag information

📹 Demo

Usage Flow

Step 1: Preparation

Connect webcam and verify recording readiness

Step 2: Start Recording

Press "REC" button to start recording

Step 3: Scan Tags

Scan QR code or barcode to mark segment

Step 4: Auto Segmentation

Segment to next tag is automatically divided

🏷️ Tags Tab

Manage scanned QR codes and barcodes, and record metadata for each tag.

Tag Information Management

  • Tag Code: Scanned QR code or barcode number
  • Scan Time: Exact time tag was recognized
  • Segment Info: Video length from previous tag to current tag
  • Notes: Record additional information like shipping status
  • Status: Display status like normal, damaged, caution

Tag Usage

Connect to Tracking Number: Link each tag to shipping tracking number for traceability
Segment Search: Quickly find and play video of specific shipment
Data Export: Export tag information and video together

📹 Videos Tab

Search, play, and download video clips segmented based on tags.

Video Management Features

  • Clip List: Display all recorded video clips in list
  • Search Filter: Search by date, tracking number, tag
  • Playback: Play video directly in browser
  • Download: Save clips locally
  • Share: Generate clip link for sharing
  • Delete: Delete unnecessary clips to free storage

Video Information

  • Video length
  • Creation date and time
  • File size
  • Related tag information
  • Playback count

⚙️ Settings Tab

Personalize and optimize how QRVideo operates.

Recording Settings

  • Resolution: Choose from 720p, 1080p, 2K
  • Frame Rate: 24fps, 30fps, 60fps

Storage Settings

  • Save Location: Local save or cloud sync

  • Storage Management: Auto-delete old files rule
  • Encryption: Encryption option for saved files

User Profile

  • Name: Enter operator name
  • Team/Department: Organization information
  • ID/Account: Operator identification info
  • Language: Korean, English, Chinese, etc.
  • Dark Mode: UI theme setting

Notification Settings

  • Large file deletion confirmation
