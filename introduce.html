📦 배송 증빙 🏷️ QR/바코드 태깅 📹 자동 녹화

QRVideo란?

QRVideo는 쇼핑몰의 배송 박스 포장 과정을 효율적으로 기록하고 관리하는 프론트엔드 웹 앱입니다.

🎯 핵심 기능

📷 실시간 녹화

카메라를 통해 배송 박스 포장 과정을 실시간으로 녹화합니다.

🏷️ 태그 기반 분할

QR코드/바코드 스캔 시점부터 다음 태그까지의 영상을 자동으로 분할합니다.

📹 클립 추출

필요한 구간의 비디오 클립을 추출하고 공유할 수 있습니다.

💾 안전한 저장

모든 기록을 체계적으로 관리하고 언제든 검토할 수 있습니다.

❓ 왜 QRVideo인가?

🛡️ 배송 분쟁 대비: 블랙 컨슈머에 의한 클레임이나 배송 손상 시 객관적인 영상 증거를 제공합니다.
⏱️ 시간 절약: 수동 기록 대신 자동으로 영상이 저장되고 구분됩니다.
📊 데이터 관리: 송장번호와 연결된 메타데이터로 체계적인 관리가 가능합니다.

📷 Camera 탭

실시간 카메라 스트림을 통해 배송 박스 포장 과정을 감시하고 녹화합니다.

주요 기능

  • 실시간 스트림: 웹캠을 통한 실시간 비디오 피드
  • 녹화 시작/중지: 한 번의 클릭으로 녹화 제어
  • QR/바코드 스캔: 태그 인식 및 자동 마킹
  • 화면 표시: 현재 상태, 경과 시간, 태그 정보 표시

📹 데모

사용 흐름

1단계: 준비

웹캠 연결 및 녹화 준비 상태 확인

2단계: 녹화 시작

"REC" 버튼을 눌러 녹화 시작

3단계: 태그 스캔

QR코드 또는 바코드 스캔하여 구간 마킹

4단계: 자동 분할

다음 태그까지의 구간이 자동으로 분할됨

🏷️ Tags 탭

스캔된 QR코드와 바코드를 관리하고, 각 태그별 메타데이터를 기록합니다.

태그 정보 관리

  • 태그 코드: 스캔된 QR코드 또는 바코드 번호
  • 스캔 시간: 태그를 인식한 정확한 시간
  • 구간 정보: 이전 태그부터 현재 태그까지의 비디오 길이
  • 메모: 배송 상태, 주의사항 등 추가 정보 기록
  • 상태: 정상, 손상, 주의 등 현황 표시

태그 활용

송장번호 연결: 각 태그를 배송 송장번호와 연결하여 추적 가능
구간별 검색: 특정 송장의 영상을 빠르게 찾아 재생
데이터 내보내기: 태그 정보와 영상을 함께 내보낼 수 있음

📹 Videos 탭

태그 기반으로 분할된 비디오 클립을 검색, 재생, 다운로드합니다.

비디오 관리 기능

  • 클립 목록: 모든 녹화된 비디오 클립을 목록으로 표시
  • 검색 필터: 날짜, 송장번호, 태그로 검색
  • 재생: 브라우저에서 직접 비디오 재생
  • 다운로드: 클립을 로컬에 저장
  • 공유: 클립 링크를 생성하여 공유
  • 삭제: 불필요한 클립 삭제로 저장공간 확보

비디오 정보

  • 영상 길이
  • 생성 날짜 및 시간
  • 파일 크기
  • 관련 태그 정보
  • 재생 횟수

⚙️ Settings 탭

QRVideo의 동작 방식을 개인화하고 최적화합니다.

녹화 설정

  • 해상도: 720p, 1080p, 2K 중 선택
  • 프레임률: 24fps, 30fps, 60fps
  • 비트레이트: 저, 중, 고 품질 설정
  • 자동 녹화: 웹캠 감지 시 자동 시작

저장소 설정

  • 저장 위치: 로컬 저장 또는 클라우드 연동
  • 자동 백업: 주기적 백업 설정
  • 저장공간 관리: 오래된 파일 자동 삭제 규칙
  • 암호화: 저장 파일 암호화 옵션

사용자 프로필

  • 이름: 작업자 이름 입력
  • 팀/부서: 소속 정보
  • ID/계정: 작업자 식별 정보
  • 언어: 한국어, 영어, 중국어 등
  • 다크 모드: UI 테마 설정

알림 설정

  • 녹화 오류 알림
  • 저장공간 부족 경고
  • 큰 파일 삭제 확인

💡 활용 사례

QRVideo가 실제 배송 현장에서 어떻게 활용되는지 살펴봅시다.

📌 사례 1: 블랙 컨슈머 대응

상황: 고객이 "박스가 손상된 상태로 도착했다"고 클레임
해결: QRVideo에 기록된 포장 영상으로 배송 전 상태를 증명

📌 사례 2: 배송 프로세스 최적화

상황: 포장 시간이 오래 걸리는 상품 확인
해결: 태그별 영상을 분석하여 병목 지점 파악 및 개선

📌 사례 3: 직원 교육

상황: 신입 직원의 포장 방법 점검
해결: 실제 포장 영상을 통한 실시간 피드백 및 개선

📌 사례 4: 품질 관리

상황: 상품별 포장 기준 일관성 유지
해결: 모든 포장 과정을 기록하여 표준화 유지
